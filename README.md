[FLCleaner 3.0][flcleaner.com] is a **stack manager**, software designed to optimize computing systems.

Whilst working similarly to other PC optimization tools, it has a distinct difference in that it works on *platform* level, rather than system. Its purpose is to help system administrators manage their [solution stack](https://en.wikipedia.org/wiki/Solution_stack) in real time â†´
